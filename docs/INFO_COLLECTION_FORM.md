## 🚀 下一步

信息收集完成后：

```bash
# 1. 创建配置
python create_shop.py --shop [店铺代码] --name "[店铺名称]"

# 2. 编辑配置文件
vim configs/shops/[店铺代码].yaml

# 3. 设置环境变量
vim .env

# 4. 验证配置
python validate_config.py --shop [店铺代码]

# 5. 开始使用
python main.py --shop [店铺代码] --task process_orders
```

---

## 💡 提示

**可共享的配置：**
- 飞书 App ID 和 App Secret（一个应用管理多个店铺）
- 物流商配置（多个店铺用同一个物流账号）
- AI配置（多个店铺用同一个OpenAI账号）
- IOSS代码（多个店铺用同一个IOSS）

**必须独立的配置：**
- 店铺代码和名称
- Etsy Shop ID
- 飞书 Table ID（每个店铺一个表格）
- 邮箱地址和密码（建议每个店铺独立）
