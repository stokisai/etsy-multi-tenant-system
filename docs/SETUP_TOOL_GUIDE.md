# 交互式配置工具使用指南

## 🎯 目标

**你只需要提供信息，工具自动生成配置，立即可用！**

不需要手动编辑配置文件，不需要理解YAML格式，只需要回答几个问题。

---

## 🚀 快速开始

### 运行配置工具

```bash
python setup_shop.py
```

工具会引导你完成所有配置。

---

## 📋 你需要准备的信息

### 每个店铺必需（3项）

1. **店铺代码**（英文）- 例如：`ocean_breeze`
2. **店铺名称**（中文）- 例如：`海洋微风店铺`
3. **Etsy Shop ID** - 例如：`12345678`
4. **飞书 Table ID** - 例如：`tblXXXXXX`
5. **邮箱地址** - 例如：`ocean@yahoo.com`
6. **邮箱密码** - 应用专用密码（用于IMAP抓取订单）

### 首次配置还需要（共享配置）

7. **飞书 App ID 和 Secret** - 所有店铺共用
8. **云途物流配置** - Customer ID, API Key, API Secret
9. **泰嘉物流配置**（可选）- Client ID, Auth Token
10. **发件人信息** - 姓名、地址、电话等

## ❓ 常见问题

### Q1: 邮箱密码是什么？

**A:** 不是你的邮箱登录密码，而是**应用专用密码**。

**Yahoo邮箱：**
1. 登录Yahoo账户安全设置
2. 生成应用专用密码
3. 使用生成的密码

**Gmail：**
1. 开启两步验证
2. 生成应用专用密码
3. 使用生成的密码

### Q2: 为什么需要邮箱密码？

**A:** 系统使用IMAP协议从邮箱抓取Etsy订单通知邮件，自动解析订单信息。

### Q3: 共享配置是什么意思？

**A:** 多个店铺可以使用相同的配置，例如：
- 飞书应用（一个应用管理多个店铺）
- 物流商账号（多个店铺用同一个物流账号）
- 发件人信息（多个店铺从同一地址发货）

### Q4: 我可以跳过某些配置吗？

**A:** 必需配置不能跳过，可选配置可以跳过：

**必需：**
- 店铺基本信息
- 飞书 Table ID
- 邮箱配置
- 云途物流配置

**可选：**
- 泰嘉物流配置
- AI配置

### Q5: 配置错了怎么办？

**A:** 重新运行配置工具，或手动编辑配置文件：

```bash
vim configs/shops/shop_code.yaml
```

### Q6: Etsy API Key在哪里？

**A:** 目前暂时不需要配置Etsy API Key。配置文件中会预留位置，未来需要时再配置。
