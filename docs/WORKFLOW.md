## 🔄 后续开新店铺（5分钟）

当你已经配置过第一个店铺后，开新店铺只需要：

```
┌─────────────────────────────────────────────────────────────┐
│  1. 收集店铺特定信息（2分钟）                                 │
│     • 店铺代码和名称                                          │
│     • Etsy Shop ID                                           │
│     • 飞书 Table ID                                          │
│     • 邮箱地址                                                │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  2. 创建配置（1分钟）                                         │
│     $ python create_shop.py --shop new_shop --name "新店铺"  │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  3. 编辑配置（1分钟）                                         │
│     只需修改4个字段：shop_id, table_id, email, sender       │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  4. 验证和运行（1分钟）                                       │
│     $ python validate_config.py --shop new_shop              │
│     $ python main.py --shop new_shop --task process_orders   │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
                         🎉 完成！
```

## 🎯 决策树：我需要准备什么？

```
开新店铺
    │
    ├─ 是第一个店铺？
    │   │
    │   ├─ 是 → 需要准备：
    │   │       • 所有必需信息（7项）
    │   │       • 共享配置（飞书、物流商、AI）
    │   │       • 时间：约20分钟
    │   │
    │   └─ 否 → 只需准备：
    │           • 店铺特定信息（4项）
    │           • 时间：约5分钟
    │
    └─ 使用相同的物流商？
        │
        ├─ 是 → 不需要重新配置物流商
        │
        └─ 否 → 需要添加新物流商配置
```

## 📋 快速检查清单

开新店铺前，确认你有：

**必需（每个店铺）：**
- [ ] 店铺代码（英文）
- [ ] 店铺名称（中文）
- [ ] Etsy Shop ID
- [ ] 飞书 Table ID
- [ ] 邮箱地址和密码

**首次需要（可共享）：**
- [ ] Etsy API Key
- [ ] 飞书 App ID 和 Secret
- [ ] 物流商配置（云途/泰嘉）
- [ ] 发件人信息
- [ ] OpenAI API Key（可选）
